## 模块职责

`collia` 的主要角色是：Agent runtime: 主机/容器采集、HTTP 监控上报、Docker 控制和反向 tunnel Service。

- 在目标主机侧执行采集和控制动作，通过 HTTP report client 上报监控批次，通过反向 tunnel 服务把容器、文件和系统控制能力暴露给 Server。
